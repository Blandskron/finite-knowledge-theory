# 🛡️ Suite de Validación Oficial (docs/validation/)

Este directorio contiene los conjuntos de datos, entradas de prueba y salidas esperadas que forman la suite de validación agnóstica de FKT.

## Propósito
Cualquier implementación de FKT (ya sea en Python, C++, Rust, CUDA, etc.) debe pasar los mismos casos de prueba de este directorio para ser considerada una implementación de referencia válida.
