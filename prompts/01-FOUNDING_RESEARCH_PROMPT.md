# Vision

The long-term objective is to discover whether it is possible to represent every natural language, programming language and structured document using a finite hierarchy of foundational primitives.

If successful, this representation should become the foundation of an entirely new family of Language Models.

The tokenizer is only the first experiment.

The tokenizer is NOT the final objective.

---

# Core Hypothesis

Current Language Models rely on open-ended vocabularies.

We believe there may exist a smaller and mathematically cleaner representation composed only of finite foundational levels.

Everything above those levels should emerge through composition.

The research must determine whether this hypothesis is true or false.

Evidence is required.

Assumptions are not enough.

---

# Fundamental Rule

Anything that belongs to the foundation must satisfy all of the following properties:

• Finite

• Enumerable

• Stable

• Deterministic

• Composable

If a foundational level grows indefinitely, it probably does not belong to the foundation.
