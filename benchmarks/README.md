# 📊 Pruebas de Rendimiento (benchmarks/)

Este directorio contiene scripts y conjuntos de datos para medir el rendimiento (velocidad, uso de memoria, compresión) de las distintas implementaciones de FKT.

## Métricas Clave
1.  **Correctness:** Coherencia de los tokens respecto a la especificación.
2.  **Throughput:** Megabytes de texto tokenizados por segundo.
3.  **Memory Peak:** Consumo máximo de memoria RAM durante cargas pesadas.
4.  **Compression Ratio:** Eficiencia de representación comparada con tokenizers tradicionales (BPE).
