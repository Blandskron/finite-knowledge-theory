# 📦 Implementación de Referencia (reference-implementation/)

Este directorio contiene el código fuente de las implementaciones oficiales de FKT.

## 🧭 Relación de Implementaciones
*   **Implementación Canónica:** Ubicada en `reference-implementation/python/` (Python sirve como especificación ejecutable, priorizando la legibilidad sobre la optimización).
*   **Implementaciones Optimizadas:** Ubicadas en carpetas separadas (ej. `reference-implementation/rust/`, `reference-implementation/cpp/`).
