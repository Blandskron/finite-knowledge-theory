# 🧪 Experimentos y Simulaciones (experiments/)

Este directorio contiene las pruebas de concepto, simulaciones matemáticas y experimentos empíricos diseñados para validar las hipótesis de FKT.

## 🧭 Principio de Reproducibilidad
Cualquier experimento almacenado aquí debe ser totalmente reproducible:
1.  **Código Limpio:** Los scripts deben guardarse en carpetas específicas (ej. `experiments/exp-001-entropy-calc/`).
2.  **Semilla Aleatoria (Seed):** Toda simulación que dependa de números aleatorios debe fijar una semilla para garantizar resultados idénticos.
3.  **Logs y Reportes:** Los resultados finales deben resumirse en un markdown local, referenciando qué hipótesis de investigación se está validando.
